Google Reader Replacement (follow up)

September 19, 2013

I spent a bit of time today customising my RSSOwl setup to make it more to my liking. I now have it looking like this:

That’s one long reader with all my feeds. Here’s how I did it.

  1. Select View -> Layout -> Newspaper
  2. Deselect View -> Toolbars -> Feed Toolbar
  3. Deselect View -> Status
  4. Right Click Main Toolbar and select Customise Toolbar
  5. Delete everything except New, Update All, Mark Read, Mark All Read, Search (or whatever is your preference)
  6. Right Click Main Toolbar and select Lock Toolbar
  7. Click Search, do a search for all titles not containing something impossible (eg asjdlfjkasdfkdaj)
  8. Save search (I called mine “All News”)
  9. Select the newly created “All News” category from your feeds (More details on this here: http://tutorial.rssowl.org/ht_searchagg.html)

Not exhaustive instructions by any means, but should get you started if you wish to go the same route.
